# 如何透過 ChatGPT 使用 MCP

> MCP 目前在 ChatGPT 仍為測試階段，請先開啟「開發者模式」來使用。

## 1. 開啟 ChatGPT，點選左下角頭像，並進入「設定」

## 2. 從設定選單中找到「應用程式」，並點選「進階設定」

> ![](/2026/img/mcp/S1.webp)

## 3. 啟用「開發者模式」，並點選「建立應用程式」

> ![](/2026/img/mcp/S2.webp)

## 4. 填入 MCP Server 的資訊並建立應用程式
